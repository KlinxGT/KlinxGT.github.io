Team Sketches: [title]

The team sketch includes the screens for ordering parts(ordering and tracking), chats with repairmen, guides, and my parts includes finding a store.

Team Paper Prototype:

The paper prototype includes the three functions of the app: diagnostics, finding parts, finding repair shop. The Diagnostics screen asks the user some questions about the issue, the order parts is similar to amazon where you can search up any parts and track the shipping, and the repair shop works like google and gives you location, ratings, prices, etc.

Team Hi-Fi Prototype: [title]

This prototype includes the start screen, each of the options(diagnostics,get help, order part), as well as a way to get back to the start screen. Diagnostics prompts the use with questions to figure out the problem, get help finds a local repairmen to chat with, and order a part allows the user to pick out parts and order it.
